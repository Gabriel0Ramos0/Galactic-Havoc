# Galactic-Havoc
Galactic Havoc é a evolução do antigo CAOS Cósmico – Missão Estelar, agora reescrito em React Native + Expo para proporcionar uma experiência nativa em dispositivos móveis. O projeto combina aventura espacial, controle via joystick virtual e ranking online de jogadores, mantendo a essência caótica e cósmica do jogo original.
